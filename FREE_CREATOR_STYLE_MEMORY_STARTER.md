# Creator Style Memory Starter

AIに毎回同じ説明をしなくてよくするための、無料スタイル記憶テンプレです。

これはAgent Memories向けの最初の無料お試しテンプレです。自動で記憶し続ける機能ではありません。まずは1枚の手動テンプレとして、自分の書き方をAIに渡せる形にします。

## 使い方

1. 下の項目を自分用に埋める
2. AIに文章を書かせる前に、このテンプレを貼る
3. 出てきた文章を見て、次回も使いたい修正点を追記する
4. ほしい拡張があればIssueで送る

Issueで送る:
https://github.com/Miraigent/Miraigent-agent-memories-mcp-catalog/issues/new?template=use_case.md

## 30秒で試す

まずは全部を埋めなくても大丈夫です。

次の4つだけ書いて、AIに貼ってください。

```text
基本トーン:
避けたい言葉:
好きな構成:
今回のテーマ:
```

例:

```text
基本トーン: 丁寧だけど堅すぎない。大げさにしない。
避けたい言葉: 革新的、完全自動、絶対、皆さん。
好きな構成: 痛み -> 原因 -> 小さな解決策 -> 無料で試せるもの -> 次の行動。
今回のテーマ: AIに毎回同じ投稿スタイルを説明している人向けの無料テンプレ紹介。
```

これだけでも、毎回ゼロから口調を説明するより、下書きのズレを減らしやすくなります。

## Before / After サンプル

同じテーマでも、スタイルメモがないとAIっぽい汎用文になりやすいです。

Before:

```text
革新的なAI活用で、あなたの発信を次のステージへ。今こそ効率化を始めましょう。
```

Style Memoryあり:

```text
AIの下書きが毎回ズレるなら、まず自分のNG表現とよく使う構成を1枚に残すと楽です。
毎回プロンプトを頑張るより、直し方を先に渡す方が早いです。
```

目的は、派手な文章にすることではありません。
いつもの自分に近い下書きを、最初から出しやすくすることです。

## 次回に残す修正メモ例

```text
直した内容:
「革新的」「次のステージへ」が大げさだった。

次回の指示:
便利さを言う時は、誇張せず「楽になる」「減らせる」「試せる」を使う。
```

この修正メモを1つずつ増やすと、自分のスタイルメモが育ちます。

## Free版の範囲

できること:
- 自分の投稿スタイルを1枚に整理する
- AIに貼って下書きのズレを減らす
- 修正メモを残す

できないこと:
- 自動で記憶し続ける
- 投稿ごとの反応を自動分析する
- X、note、blogごとに自動最適化する
- チームや複数アカウントで管理する

反応があれば、Pro PackやAgent Memories側で用途別テンプレ、投稿後Reflection、継続記憶の連携候補へ広げます。

## 試したあとに見ること

1回使ったら、次の3つだけ確認してください。

- 直す回数が減ったか
- 自分が使わない言葉が減ったか
- 次回も残したい修正メモが1つ増えたか

この3つが当てはまる場合、Agent Memoriesで継続記憶にする価値がある候補です。
