# X Japanese Business Auto Post MCP - Implementation Handoff

Owner: ConoHa VPS implementation members

Purpose: create the first high-impact free MCP for Miraigent / Agent Memories
distribution.

## Goal

Build a public MCP that helps Japanese creators, solo founders, and small
business operators stop thinking about every X post one by one.

This is the first acquisition-focused MCP. The target is not a tiny demo. The
tool should be useful enough to promote publicly and support the 10,000
impressions goal.

The BtoC promise is:

- enter one theme
- get a week of posts
- approve the plan
- let the tool publish or prepare the posts
- receive a simple improvement memo

People want less work. The tool must sell "automation and effort reduction",
not only better writing assistance.

## Free Version Scope

The first public version should include:

- one-theme-to-7-days posting plan
- Japanese business post generation
- X thread generation
- batch post generation from one input
- CTA and hashtag suggestions
- tone and NG expression check
- risk check before posting
- batch approval screen/output
- scheduled posting or posting packet through official or user-confirmed flow
- daily/weekly post result summary
- next-post improvement memo
- reusable posting profile for the next batch
- Agent Memories memory candidates:
  - tone
  - audience
  - CTA
  - NG expressions
  - weekly theme
  - improvement notes

## Out Of Scope For First Release

- bypassing login, CAPTCHA, two-factor authentication, or platform limits
- silent posting with no setup approval or review mode
- storing X credentials in the repository
- scraping private data
- advanced long-term reaction analysis
- multi-account automation
- image/video posting
- paid team approval workflows

## Recommended MCP Tools

Action-oriented tool candidates:

- generate_x_post
- generate_x_thread

- approve_x_batch
- review_x_post
- prepare_x_publish
- schedule_x_batch
- record_x_result

- extract_agent_memory_candidates

## Public Positioning

Use strong but defensible language:

- Japanese business X posting MCP
- turns one theme into a week of X posts
- drafts, checks, schedules/prepares, and improves posts
- built for creators, solo founders, and small business operators
- reduces the daily work of thinking up posts
- connects naturally to Agent Memories for remembering tone and improvements

Avoid claims unless rechecked at release time:

- world first
- only one
- no competitors
- guaranteed growth
- guaranteed impressions

## Done Definition

The first release is ready when:

- public repo or package exists
- README explains setup and one clear use case
- sample input/output is included
- no secrets or credentials are committed
- setup approval or batch confirmation is visible
- one input can produce a 7-day posting packet
- README shows how it reduces daily posting work
- at least one demo output can be shared on X/note
- catalog links are updated
- Miraigent resources/blog can link to it

## Distribution Plan

After release:

1. Update this catalog.
2. Publish a Miraigent blog/note article explaining the use case.
3. Post an X thread with a concrete before/after example: "from one theme to 7 days of posts".
4. Ask for Issues/use cases from creators and solo founders.
5. Track impressions, clicks, stars, issues, and Agent Memories signups.
