# Research-Based Action MCP Candidates

This document tracks high-impact MCP candidates for Miraigent / Agent Memories.

Goal: publish free tools that attract users, not only small demos.

The candidates below focus on action-oriented MCPs:

- create
- post
- submit
- scan
- track
- improve

They are stronger acquisition assets than passive research-only tools.

## Important Release Rule

Be aggressive with product value, but conservative with claims.

- Do not claim "world first", "only one", or "competitor zero" in public release
  copy unless checked again at release time.
- Do not bypass login, CAPTCHA, two-factor authentication, rate limits, or
  platform rules.
- For posting, applications, press release submission, or affiliate program
  application, keep a human confirmation step unless an official API and safe
  account flow are confirmed.
- Free tools should still deliver a strong output. Paid/Agent Memories value
  should come from memory, continuity, personalization, history, and long-term
  improvement.

## S Priority: Acquisition Flagship Tools

### S-1. note Auto Publishing MCP

Why it matters:

- note is a major Japanese publishing and paid content platform.
- A note-focused MCP is a strong differentiator for Japanese creators,
  educators, indie makers, and small businesses.
- Agent Memories can connect naturally through writing style, article history,
  tags, paid/free boundary notes, and post-publication improvement memory.

Free flagship scope:

- Create note-ready article body.
- Format headings, link cards, hashtags, and CTA.
- Prepare thumbnail/eyecatch instructions.
- Run pre-publish checks.
- Create or update a draft when a safe authenticated route is available.
- Keep human confirmation before publish when account, CAPTCHA, or platform
  state is uncertain.

Agent Memories expansion:

- Remember writing style, article series, tags, CTA patterns, product boundary,
  and revision lessons.
- Store post URL, update history, reaction notes, and next article ideas.

Press angle:

- Japanese note publishing workflow MCP for creators and small businesses.

### S-2. X Japanese Business Auto Post MCP

Why it matters:

- X posting automation has clear demand.
- Japanese business use cases are underserved compared with English developer
  tooling.
- It can drive visible distribution for every Miraigent / Agent Memories asset.

Free flagship scope:

- Generate Japanese business posts, threads, and replies.
- Add pre-post risk and tone checks.
- Optimize hook, CTA, hashtags, and thread order.
- Post through official or safe user-confirmed flow where available.
- Save a local result summary.

Agent Memories expansion:

- Remember winning hooks, NG expressions, audience, product CTA, and post
  performance notes.
- Improve future posts from saved reaction memory.

Press angle:

- Japanese business posting MCP with human review and memory-based improvement.

### S-3. YouTube to Multi-Channel Repurpose MCP

Why it matters:

- Creators and businesses need to turn one video into many assets.
- It connects naturally to X, note, blog, LinkedIn, and short-form captions.

Free flagship scope:

- Input a YouTube URL or transcript.
- Generate X thread, note outline, blog outline, LinkedIn post, and short video
  caption.
- Extract title, key points, quotes, and CTA suggestions.
- Export publish-ready files.

Agent Memories expansion:

- Remember channel tone, previous series, audience, CTA, and what performed
  well.

Press angle:

- One video to Japanese multi-channel publishing MCP.

### S-4. Agent Memory Launch Pack MCP

Why it matters:

- Product launches need README, npm copy, note, X, FAQ, and press draft at once.
- It can be used for every future free MCP release.

Free flagship scope:

- Generate GitHub README draft.
- Generate npm package description.
- Generate note structure.
- Generate X launch thread.
- Generate SEO blog outline.
- Generate FAQ.
- Generate press release draft.

Agent Memories expansion:

- Store launch context, positioning, reactions, issues, and next launch
  improvements.

Press angle:

- Launch workflow MCP for indie makers, creators, and AI tool builders.

## A Priority: Revenue/Operations Tools

### A-1. Japanese Affiliate Program Scanner MCP

Free flagship scope:

- Scan public or user-provided affiliate program information.
- Score fit by niche, commission, approval conditions, and content match.
- Generate application preparation notes.
- Keep application submission human-confirmed.

Guardrails:

- Do not auto-submit applications without user confirmation.
- Do not scrape behind login or bypass platform restrictions.
- Do not store affiliate credentials.

### A-2. Affiliate Link Management MCP

Free flagship scope:

- Organize affiliate links and campaign notes.
- Generate link inventory, content placement plan, and tracking checklist.
- Export CSV/Markdown.

Agent Memories expansion:

- Remember product fit, content performance, and campaign learnings.

### A-3. SEO Article to WordPress MCP

Free flagship scope:

- Generate SEO article outline and draft.
- Prepare WordPress-ready Markdown/HTML.
- Suggest category, tags, FAQ, and internal links.
- Draft publish package.

Guardrails:

- Publishing requires official API credentials and user confirmation.

### A-4. GitHub Trending Japanese Explainer MCP

Free flagship scope:

- Fetch or accept GitHub trending repo data.
- Generate Japanese explanation, X thread, note outline, and business angle.
- Export source links and caution notes.

Why it matters:

- Strong for engineer discovery and social sharing.

## B Priority: Media/PR Tools

### B-1. Press Release Draft and Submission Prep MCP

Free flagship scope:

## Free vs Paid Principle

Free:

- Strong one-shot output.
- Exportable files.
- Human confirmation gates.
- Public-safe memory candidates.

Agent Memories / paid:

- Long-term memory.
- Post-publication improvement.
- Multi-account or multi-project continuity.
- Historical reaction analysis.
- Personalization across launches and posts.
- Team workflow and approval history.
