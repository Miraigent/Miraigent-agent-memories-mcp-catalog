# Research-Based Action MCP Candidates

This document tracks high-impact MCP candidates for Miraigent / Agent Memories.

Goal: publish free tools that attract users, not only small demos.

The candidates below focus on action-oriented MCPs:

- create
- post
- submit
- scan
- track
- improve

They are stronger acquisition assets than passive research-only tools.

## Important Release Rule

Be aggressive with product value, but conservative with claims.

- Do not claim "world first", "only one", or "competitor zero" in public release
  copy unless checked again at release time.
- Do not bypass login, CAPTCHA, two-factor authentication, rate limits, or
  platform rules.
- For posting, applications, press release submission, or affiliate program
  application, keep a human confirmation step unless an official API and safe
  account flow are confirmed.
- Free tools should still deliver a strong output. Paid/Agent Memories value
  should come from memory, continuity, personalization, history, and long-term
  improvement.

## S Priority: Acquisition Flagship Tools

### S-1. note Auto Publishing MCP

Why it matters:

- note is a major Japanese publishing and paid content platform.
- A note-focused MCP is a strong differentiator for Japanese creators,
  educators, indie makers, and small businesses.
- Agent Memories can connect naturally through writing style, article history,
  tags, paid/free boundary notes, and post-publication improvement memory.

Free flagship scope:

- Create note-ready article body.
- Format headings, link cards, hashtags, and CTA.
- Prepare thumbnail/eyecatch instructions.
- Run pre-publish checks.
- Create or update a draft when a safe authenticated route is available.
- Keep human confirmation before publish when account, CAPTCHA, or platform
  state is uncertain.

Agent Memories expansion:

- Remember writing style, article series, tags, CTA patterns, product boundary,
  and revision lessons.
- Store post URL, update history, reaction notes, and next article ideas.

Press angle:

- Japanese note publishing workflow MCP for creators and small businesses.

### S-2. Tips Article Publishing MCP

Why it matters:

- Tips is a Japanese paid-content and creator-commerce platform.
- Official guidance exists, but many creators still struggle to connect article
  creation, formatting, cover images, sales links, and publishing operations.
- This fits both Agent Memories and Gasおじ-style tool distribution: publish a
  useful free helper first, then expand into remembered writing style, product
  boundaries, and launch workflows.

Free flagship scope:

- Create Tips-ready article body.
- Prepare title, lead, paid/free boundary notes, CTA, and tag suggestions.
- Prepare cover/eyecatch checklist.
- Generate a publish checklist and manual handoff.
- Create or update a draft only when a safe authenticated route and platform
  rules are confirmed.
- Keep human confirmation before publish.

Agent Memories expansion:

- Remember product positioning, article style, price boundary, CTA pattern,
  reader questions, and revision lessons.
- Store article URL, update history, sales/reaction notes, and next launch
  ideas.

Gasおじ expansion:

- GAS utilities can use the same distribution philosophy: free useful tool,
  clear docs, visible examples, and upgrade path through templates, paid
  articles, or support.

Press angle:

- Tips article workflow MCP for Japanese creators and paid content sellers.

### S-3. X Japanese Business Auto Post MCP

Why it matters:

- X posting automation has clear demand.
- Japanese business use cases are underserved compared with English developer
  tooling.
- It can drive visible distribution for every Miraigent / Agent Memories asset.
- BtoC users want to reduce repetitive work. A strong X MCP should sell
  "one theme becomes a week of posts" rather than only "better post drafts".

Free flagship scope:

- Generate a 7-day X posting plan from one theme.
- Generate Japanese business posts, threads, and replies in batch.
- Add pre-post risk and tone checks.
- Optimize hook, CTA, hashtags, and thread order.
- Let the user approve the batch once.
- Schedule or prepare posts through official or safe user-confirmed flow where
  available.
- Save a daily/weekly result summary.
- Suggest the next batch from the results.

Agent Memories expansion:

- Remember winning hooks, NG expressions, audience, product CTA, and post
  performance notes.
- Improve future posts from saved reaction memory.

Press angle:

- Japanese business posting MCP with human review and memory-based improvement.

### S-4. Multi-SNS Publishing Prep MCP

Why it matters:

- Users do not want one isolated posting helper. They want X, Threads, Bluesky,
  Instagram, LinkedIn, note, Tips, Zenn, and blogs to work as one publishing
  flow.
- Even if some platforms require manual confirmation, a unified preparation
  MCP is valuable because it creates platform-specific copy and checklists.

Free flagship scope:

- Generate platform-specific posts from one source idea.
- Prepare X thread, Threads post, Bluesky post, Instagram caption, LinkedIn
  post, note outline, Tips outline, and blog outline.
- Add platform-specific warnings, length checks, and CTA variations.
- Export a publishing packet.
- Keep actual posting human-confirmed unless official/safe posting routes are
  confirmed.

Agent Memories expansion:

- Remember platform-specific tone, audience, hashtag style, CTA, and past
  performance notes.

Press angle:

- Japanese multi-platform creator publishing MCP.

Press angle:

## Free vs Paid Principle

Free:

- Strong one-shot output.
- Exportable files.
- Human confirmation gates.
- Public-safe memory candidates.

Agent Memories / paid:

- Long-term memory.
- Post-publication improvement.
- Multi-account or multi-project continuity.
- Historical reaction analysis.
- Personalization across launches and posts.
- Team workflow and approval history.
