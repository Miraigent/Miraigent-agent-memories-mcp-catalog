# Issue投稿ルール

このrepoでは、Agent Memories向けMCP候補の要望や利用例をIssueで集めます。

## 歓迎するIssue

- 欲しいMCP候補
- 入力項目の提案
- 出力項目の提案
- 無料で試したい候補
- Agent Memories契約者向けに欲しい候補
- クリエイター/個人事業主向けの利用例

## 書かないでください

- API key、token、cookie、private key
- 個人情報
- 顧客情報
- 非公開の業務マニュアル
- MIRAI Memory本体の内部仕様
- working memory MCPの実装依頼
- 医療/法律/税務など専門家判断の代替依頼

## Issueを書く時の型

- 欲しいMCP名
- 誰が使うか
- 何を入力するか
- 何を出力してほしいか
- 無料/有料/契約者向けのどれがよいか
- Agent Memoriesでどう記憶に残したいか
