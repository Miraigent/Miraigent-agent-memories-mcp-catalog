# MCP候補一覧

この一覧は、Agent Memoriesで使う予定のMCP候補です。

候補は「無料で試せるもの」「Agent Memories契約者向けに育てるもの」「Miraigentの業務導線へつなぐもの」に分けて検討します。

## 優先度A: 初回無料候補

### 0. Auto Post & Improve Starter MCP

対象:

- X、note、ブログで発信を継続したい個人/クリエイター

目的:

- 投稿を作るだけでなく、投稿実行、投稿後記録、次回改善まで回す
- 無料版はテキスト投稿と最低限の改善に絞る
- 初回だけの有料級無料配布としてIssueと認知を集める

入力:

- 投稿テーマ
- 投稿先
- 投稿本文または下書き
- 投稿前チェック条件
- 投稿後に残したい反応メモ

出力:

- 投稿前チェック結果
- 投稿実行結果
- 投稿URL
- 反応メモ
- 次回改善案

Agent Memories接続:

- 有料版では、投稿履歴、反応、口調、NG表現、勝ちパターンを記憶して次回投稿へ反映する
- 画像付き投稿、動画付き投稿、複数媒体投稿、定期投稿、反応分析は有料拡張にする

### 1. SNS下書きMCP

対象:

- X、Threads、Bluesky、Instagramなどで発信したい個人/クリエイター

目的:

- 投稿ネタを整理する
- 下書きを作る
- 過去の反応や自分の方針に合わせて改善する

入力:

- 投稿テーマ
- 伝えたいこと
- 投稿先SNS
- 口調
- 避けたい表現

出力:

- 投稿下書き
- 返信用の短文
- 連投構成
- 改善メモ

Agent Memories接続:

- よく使う口調、NG表現、伸びた投稿傾向を記憶する

### 2. ブログ/note補助MCP

対象:

- note、ブログ、Zenn、Qiitaで発信したい人

目的:

- メモから記事構成を作る
- タイトル案を出す
- 読者向けに文章を整える

入力:

- 書きたいテーマ
- メモ
- 読者
- 目的

出力:

- 見出し案
- 本文下書き
- 導入文
- CTA案

Agent Memories接続:

- 書き方、読者像、過去記事の反応、継続テーマを記憶する

### 3. Reflection振り返りMCP

対象:

- AIキャラクター運用者
- Agent Memories利用者
- 日々の判断や失敗を残したい人

目的:

- 今日の出来事を振り返る
- 失敗、誤解、改善点を整理する
- 次回の行動に変える

入力:

- 今日の出来事
- うまくいったこと
- 困ったこと
- 次回気をつけたいこと

出力:

- 振り返りメモ
- 次回チェック項目
- 記憶候補
- 昇格すべき/保留すべき項目

Agent Memories接続:

- 一時メモから、再利用できる運用記憶へ昇格する候補を作る

## 優先度B: 生活・学習・創作

### 4. 学習/調査MCP

目的:

- 学びたいテーマを分解する
- 調査メモを整理する
- 次に読むもの、聞くこと、試すことを決める

### 5. 生活不安ナビMCP

目的:

- 夜中の不安や困りごとを整理する
- 医療/法律など専門家判断が必要な領域では、初動整理と相談先確認に限定する

### 6. 手続き整理MCP

目的:

- 引越し、役所、保険、相続準備などの手続きを整理する
- 必要書類と相談先をリスト化する

### 7. 漫画/創作プロットMCP

目的:

- キャラクター設定、世界観、プロット、セリフ案を整理する
- 作品ごとの設定を記憶候補として残す

### 8. ショート動画台本MCP

目的:

- TikTok、YouTube Shorts、Reels向けの短い台本を作る
- フック、展開、締めを整理する

## 優先度C: Agent Memories契約者向け候補

### 9. 家族メモリーMCP

目的:

- 家族の思い出、会話、写真の説明、残したい言葉を整理する

### 10. 終活メモリーMCP

目的:

- 自分の考え、家族への言葉、手続きメモを安全に整理する

注意:

- 法務/相続判断の代替にしない
- 相談先確認とメモ整理に限定する

### 11. キャラクター記憶設定MCP

目的:

- AIキャラクターの口調、関係性、重要な出来事、避けたい表現を整理する

### 12. スキル図鑑連携MCP

目的:

- Agent Memories内で使えるスキル/MCPを一覧化し、無料/有料/契約者優遇/NFT対象を分ける

## Issueで集めたいこと

- このMCPが欲しい
- この入力項目が足りない
- この出力があると便利
- 無料で試したい
- Agent Memories契約者向けなら使いたい
- NFTや先行配布と組み合わせたい

## 次に実装候補へ進める基準

- IssueやDiscordで反応が多い
- 無料で試す価値が分かりやすい
- Agent Memories本体への導線が自然
- 個人/クリエイターが継続利用しやすい
- 非公開情報や専門家判断に踏み込みすぎない
