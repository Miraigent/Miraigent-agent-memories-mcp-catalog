# 3段階商品設計

この文書は、Auto Post & Improve系MCPを「一般無料版」「Agent Memoriesユーザー特典」「有料/上位版」の3段階で設計するための正本です。

## 全体方針

3段階の役割:

1. 一般無料版
   - 集客用
   - 「X/noteへ実際に投稿できる」で注目を取る
   - テキスト投稿実行までに絞る

2. Agent Memoriesユーザー特典
   - 登録理由を作る
   - 投稿後の記録と改善を体験してもらう
   - Agent Memoriesの「覚えて改善する」価値を無料特典として見せる

3. 有料/上位版
   - 収益化用
   - 画像、動画、複数媒体、定期投稿、長期分析を扱う
   - 本格運用、法人運用、Miraigent導線にもつなげる

## 1. 一般無料版

想定名:

- Auto Post Free
- Auto Post Starter
- Auto Post Lite

目的:

- 無料なのに「実際に投稿できる」驚きを作る
- GitHub Issueを集める
- Agent Memoriesへの登録導線を作る

含める機能:

- テキスト投稿本文の作成
- 投稿前チェック
- 人間確認ゲート
- Xテキスト投稿実行
- noteテキスト投稿実行
- 単発または少数投稿
- 規約順守と本人判断の注意書き
- 非保証、自己責任、販売対象外の境界表示

含めない機能:

- 投稿URLの自動記録
- 投稿結果の保存
- 反応メモ
- 次回改善案
- 画像投稿
- 動画投稿
- 複数媒体同時投稿
- 定期投稿
- 長期分析

見せ方:

「投稿作成ツール」ではなく「X/noteへ実際に投稿できる無料MCP」として見せる。

## 2. Agent Memoriesユーザー特典

想定名:

- Auto Post Memory Pack
- Agent Memories Posting Pack
- Memory Posting Bonus

目的:

- Agent Memories登録の理由を作る
- 投稿後の記録と改善で、記憶の価値を体験してもらう
- 有料/上位版へ自然につなげる

含める機能:

- 一般無料版の全機能
- 投稿URLの記録
- 投稿結果の記録
- 反応メモ
- 次回改善案
- 短期投稿履歴を使った改善

商品ごとに無料開放を検討する機能:

- 画像投稿
- 動画投稿
- 複数媒体投稿
- 定期投稿
- 簡易反応分析
- 口調/NG表現の簡易保存

見せ方:

「Agent Memoriesユーザーなら、投稿して終わりではなく、次に改善するところまで無料で使える」と伝える。

## 3. 有料/上位版

想定名:

- Auto Post Pro
- Auto Post & Improve Pro
- Creator Growth Automation

目的:

- 収益化する
- 本格運用ユーザーを支える
- Miraigentの法人導線にもつなげる

含める機能:

- Agent Memoriesユーザー特典の全機能
- 画像付き投稿
- 動画付き投稿
- 複数媒体同時投稿
- 定期投稿スケジュール
- シリーズ投稿
- 長期投稿履歴からの改善
- 反応分析
- 口調、NG表現、勝ちパターン保存
- 商品導線、CTA改善
- チーム承認
- 法人向け承認フロー

有料化理由:

- 画像、動画、複数媒体、継続運用は運用価値が高い
- 記録、分析、改善が積み上がるほど価値が増える
- 法人運用では安全性、承認、履歴管理が必要になる

## Miraigentとの切り分け

Miraigentに寄せるもの:

- 公式APIや明確に許可された連携
- 法人向け承認フロー
- 投稿履歴、監査、説明責任
- チーム運用
- 安全運用設計
- 「各サービス規約・API許可範囲内で利用」を明記できるもの

Agent Memories / OSS無料に寄せるもの:

- 個人クリエイター向けの便利ツール
- APIなし/許可不明の投稿補助
- 利用者本人の判断で使う実験ツール
- Issue収集と集客
- 非公式自動化は販売対象外として扱う

固定表現:

- 各サービス規約・API許可範囲内で利用してください。
- 非公式自動化は販売対象外です。
- APIなし/許可不明の投稿機能は、無料OSS/検証用であり、非保証です。
