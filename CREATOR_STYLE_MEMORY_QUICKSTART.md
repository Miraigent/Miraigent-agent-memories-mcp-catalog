# Creator Style Memory Quickstart

AIに投稿下書きを頼むたびに「その口調じゃない」と直している人向けの、1投稿だけで試す手順です。

これは自動で記憶し続ける機能ではありません。Agent Memoriesの考え方を、まず公開しても安全な手動テンプレで試すためのQuickstartです。

## 1. 30秒で4項目を書く

    基本トーン:
    避けたい言葉:
    好きな構成:
    今回のテーマ:

例:

    基本トーン: 丁寧だけど堅すぎない。大げさにしない。
    避けたい言葉: 革新的、完全自動、絶対、皆さん。
    好きな構成: 痛み -> 原因 -> 小さな解決策 -> 無料で試せるもの -> 次の行動。
    今回のテーマ: AIに毎回同じ投稿スタイルを説明している人向けの無料テンプレ紹介。

## 2. AIに貼って1投稿だけ作る

    以下のスタイルメモを守って、投稿文を1つ作ってください。

    基本トーン:
    避けたい言葉:
    好きな構成:
    今回のテーマ:

## 3. 出てきた文章を3つだけ確認する

- 自分が使わない言葉が減ったか
- 直す回数が少しでも減ったか
- 次回も残したい修正メモが1つ見つかったか

1つでも当てはまったら、スタイルメモを育てる価値があります。

## 4. 次回に残す修正メモを書く

    直した内容:
    次回は避ける表現:
    次回も使いたい構成:

例:

    直した内容: 「革新的」「次のステージへ」が大げさだった。
    次回は避ける表現: 夢のような、誰でも簡単、完全自動。
    次回も使いたい構成: 痛みを先に置いて、最後に小さな行動を出す。

## Free / Pro / Subscriber Route

Free:

- 1枚の手動テンプレ
- 1用途だけ試す
- 公開しても安全なサンプルだけを使う
- 修正メモを次回用に残す

Pro / subscriber route candidate:

- X、note、blog別テンプレ
- 投稿後Reflection
- Correction Library
- 継続記憶
- Agent Memories連携

反応が見えた用途から拡張します。まだ測れていない用途を、完成済み機能のようには扱いません。

## Safe Use

- パスワード、API key、token、cookie、private keyを書かないでください。
- 顧客情報、個人情報、支払い情報、医療・法務・税務・本人確認情報を書かないでください。
- 非公開の会話ログや未公開事業情報は、公開してよい仮例に置き換えてください。
- このテンプレは売上、フォロワー、バズ、自分の文体の完全再現を保証しません。
